Installing Python With Anaconda

Sun 20 October 2019

Installing Anaconda is as simple as downloading the installation packages for the operating system of choice and running installation file on the machine.

Download the installation file fron here, make sure you download the correct installer for the operating system that you have on your machine.

For Windows:

Once the installer has been downloaded, double click on the downloaded file to launch the installer.

The installation wizard should walk you through the installation process, here is a detailed step-by-step explanation to help you install.

For Linux:

Once the ‘.sh’ file has been downloaded, make a note of the path where the file is located.

Install all the dependencies necessary for successfully installing Anaconda using the commands;

For Ubuntu:

apt-get install libgl1-mesa-glx libegl1-mesa libxrandr2 libxrandr2 libxss1 libxcursor1 libxcomposite1 libasound2 libxi6 libxtst6

For Fedora/RedHat: 

    yum install libXcomposite libXcursor libXi libXtst libXrandr alsa-lib mesa-libEGL libXdamage mesa-libGL libXScrnSaver

Note: Depending on your local machine, you may be required to run the commands with super user permissions. In order to do so, just prefix sudo to the above commands and provide the superuser credentials when prompted.

Run the command; yes

    $> bash /path/to/installer/Anaconda3-2019.10-Linux-x86_64.sh

Note: The shell file name mentioned here is an illustration, it should be the same as the one that was downloaded.
